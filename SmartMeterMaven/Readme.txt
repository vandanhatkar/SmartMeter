This is Maven project having Springboot.

To run use following commands:

1)Go to SmartMeterMaven project directory. run following command: mvn -e install (execute unit test cases)
2)Go to SmartMeterMaven project directory. run following command: mvn -e spring-boot:run (use postman and hit Url: http://localhost:8081/api/smart/reads/4)
